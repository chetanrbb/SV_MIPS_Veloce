This folder consists of all the files required for simulation.
Checker module in this folder is different as compared to the one in the SourceCode folder.
Rest of all the design files are same.
This folder also consists of:
1. top_module: This is the top module which instantiates cpu, checker interface and checker module
2. testbench_cpu: This is the testbench module which drives top_module to generate results on Questa Simulator.
Simulation statistics are included in the presentation and was generated using "simstates" command on Questa command line.
